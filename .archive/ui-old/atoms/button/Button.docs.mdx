# Button

A polymorphic, stylable button that supports:

- Variants: `primary`, `secondary`, `ghost`, `outline`
- Sizes: `sm`, `md`, `lg`
- Intents: `info`, `danger`, `success`, `warning`
- Tones: `subtle`, `neutral`, `strong`
- Shapes: `round`, `square`, `pill`
- Elevation levels: `none`, `sm`, `md`, `lg`

---

## Usage

```tsx
<Button variant="primary" size="md" intent="info">
  Click Me
</Button>
````

## Props

| Prop        | Type              | Default     | Description                            |             |                            |                             |
| ----------- | ----------------- | ----------- | -------------------------------------- | ----------- | -------------------------- | --------------------------- |
| `variant`   | \`'primary'       | 'secondary' | 'ghost'\`                              | `'primary'` | Button color style         |                             |
| `size`      | \`'sm'            | 'md'        | 'lg'\`                                 | `'md'`      | Button size                |                             |
| `intent`    | \`'info'          | 'danger'    | 'success'                              | 'warning'\` | `undefined`                | Semantic meaning or purpose |
| `tone`      | \`'subtle'        | 'neutral'   | 'strong'\`                             | `'neutral'` | Emphasis tone              |                             |
| `shape`     | \`'round'         | 'square'    | 'pill'\`                               | `'round'`   | Button border radius style |                             |
| `elevation` | \`'none'          | 'sm'        | 'md'                                   | 'lg'\`      | `'none'`                   | Visual depth                |
| `asChild`   | `boolean`         | `false`     | Render as span or anchor (polymorphic) |             |                            |                             |
| `children`  | `React.ReactNode` | `required`  | The button content                     |             |                            |                             |
